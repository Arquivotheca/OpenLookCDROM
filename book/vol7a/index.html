XView Programming Manual

This book tells you in tutorial fashion most everything you need to know about developing X Window System applications with the XView toolkit.

Online Viewing using Adobe Acrobat

You can look at the entire book using Acrobat.

Note: if the above fails with a message like "Cannot open /tmp/abaaa4829", it is likely a Mosaic problem; use the AcroRead File Chooser to go to the books/vol3a directory and open book.pdf.

The Acrobat version has PDF "bookmarks" for each chapter and appendix (with the Acrobat Reader V1.x, you click on the icon, not the name, of each bookmarked entry). As well, each entry in the Table of Content is hyperlinked to the appropriate page. Do you need to learn about Acrobat and PDF? Or install an Acrobat Reader from the CD-ROM?
