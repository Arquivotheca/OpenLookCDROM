Welcome to the Online Tour of The OPEN LOOK and XView Source and Documentation CD-ROM!

This CD-ROM compilation contains Darwin Open Systems' OPEN LOOK and XView Source and Documentation CD-ROM (formerly called The OPEN LOOK GUI, XView and NeWS Archive). We have collected three full OPEN LOOK textbooks, the example programs from several OPEN LOOK and X11 programming texts, the latest versions of XFree86 R4, and many useful programs. And we've compiled many of them for both Linux and SunOS, so that you can run them out of the box on two of the highest-volume UNIX platforms in use today. We've also provided pre-built search indexes for Sun's SearchIt and Adobe's Acrobat Catalog searching tools to help you find specific information that you need.

We've organized this online tour using HTML to allow you to browse it with Netscape, Konqueror, or other browser If you are on the Internet, this will also allow you to follow links to the outside, which are scattered throughout the tour.

Before you start, you may need to tell your Web Viewer about certain file types. If necessary, the files in the WebHelpers directory can be added to your .mime.types and .mailcap file, or can be used to create those files. For other browswers, you may need to enter this information manually in a "Helper Applications" dialog.

Overview

Here is an approximate pictorial overview of the original CD's contents (this version has a few additional directories):

In a little more detail:

  • README - text version of this information
  • book - top-level for textbooks. There are two books from the O'Reilly X series, Volume 3, User's Guide, OPEN LOOK edition (draft), and Volume 7, XView Programming (Volume 7A, Programming Manual and Volume 7B, XView Reference). All three are online in both PostScript and PDF.
  • doc - top-level for other documents such as FAQ's
  • index.html - this file
  • indices - pre-built search indexes
  • man - top level for man pages (included in troff, catable, PostScript and PDF)
  • src - top-level for megabytes of source code
  • src/xview - XView Program source
  • src/NeWS - NeWS Program source
  • viewers - Viewers for PostScript and PDF
  • demos - top-level for commercial demos

Window Systems Source Code

Since this archive is intended to be of interest to, among others, window system afficionados, we've also included the source code for not one but two complete window systems:
  • The XFree86 version of The X Window System, Version 11.
  • The MGR Window System by Steve Uhler of Bellcore. Not at all compatible with X11: mgr is small, simple, and very fast.
