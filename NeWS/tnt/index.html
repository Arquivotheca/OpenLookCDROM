TNT Sources

Here is some of the neat stuff that was written for TNT:

  • hexsweeper, a clever Minesweeper game
  • jet, Sun's Jet terminal program
  • motnt.ps, Motif-like menus for TNT
  • pietab, makes menus come out as pie charts
  • pizzatool, demonstration program to order pizzas
  • squig, paints interesting squiggles on the root window
