Darwin Open Systems X11 Distribution

This is the complete distribution of X11 XFree86 Release 4.1. Darwin Open Systems bring you the complete X11R6 source distribution, uncompressed, untarred, and ready to work on.

Contents

  • Index.html - this file
  • xc/ - the entire XFree86 core software, unpacked
  • doctools/ - the XFree86 documentation tools, unpacked

If you wish to build the X Window System (or parts of it) from these sources, please read the Release Notes in the xc directory for information on build the software.

You can obtain later versions of this software from http://www.xfree86.org.
