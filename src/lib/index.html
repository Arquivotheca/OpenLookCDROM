XView Source

We are lucky that Sun have released the XView source code for public release. It is not "public domain", they hold the copyright to it, but you can do anything you want with it. Here we have two complete implementations of XView, plus several additional bits and pieces:

  • xview3L5, Kenneth Osterberg's package of XView libraries (and binaries) for Linux
  • xview3.2p1-X11R6-LinuxElf, a complete XView source distribution ready for compiling with ELF.
  • SlingShot additions to XView
  • UIT, the User Interface Toolkit
  • some minor XView add-ons
  • xpm, the complete X PixMap library.
