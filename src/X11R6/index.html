Darwin Open Systems X11 Distribution

This is the complete distribution of X11 Release 6. Darwin Open Systems bring you the complete X11R6 source distribution, uncompressed, untarred, and with all core ("xc") patches up to Patch Level 11 already in place.

Contents

  • Index.html - this file
  • README - The X Consortium README file (from which parts of this file are taken)
  • RELNOTES.PS - Release Notes in PostScript
  • RELNOTES.pdf - Release Notes in Adobe's Portable Document Format
  • RELNOTES.TXT - Release Notes in plain text
  • RELNOTES.ms - Release Notes, troff -ms source
  • Manifest.PS - Contrib Manifest in PostScript
  • Manifest.ms - Contrib Manifest, troff -ms source
  • UNPACK_TARS - script to convert the xc*tar and contrib*tar to usable files
  • xc/ - the entire X11R6 core software, unpacked
  • contrib/ - the entire X11R6 contributed software, unpacked
  • fixes/ - public patches to core software already applied
  • contrib-fixes/ - patches to contrib software not applied
  • xdm-auth/ - Info about HasXdmAuth config option

If you wish to build the X Window System (or parts of it) from these sources, please read the Release Notes for information on build the software.

If you wish to maintain this directory, you may want to FTP any X Consortium Public Fixes that are later than Patch 11, and apply them yourself. You may also wish to consult the file GettingR6 for information on buying printed documentation, tapes, and CD-ROMs from the X Consortium.
