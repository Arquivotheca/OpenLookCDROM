NAME
  abob - Bob control panel.

SYNOPSIS
  abob [dir]

DESCRIPTION
  Try it.

  If dir is specified, abob checks for bob.snd in that directory.  If not
  specified, and the environment variable SOUND_PATH is	defined, abob will
  check	for bob.snd in the directories specified by SOUND_PATH.

RESTRICTIONS
  Recorded Bob is only available in 8KHz mono at this time.  Watch this	space
  for an announcment about Bob in HiFi.

ENVIRONMENT

  AUDIOFILE
	  to get the default host and server to	use.

  SOUND_PATH
	  specifies a colon-separated list of directories in which sounds may
	  be found.
