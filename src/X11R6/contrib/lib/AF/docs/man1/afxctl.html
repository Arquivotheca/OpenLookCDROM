NAME
  afxctl - AF control panel

SYNOPSIS
  afxctl

DESCRIPTION
  This program is an X based control panel for an AF server.  afxctl allows
  manipulation of input	and output gain	and input and output device selection
  using	an X window control panel.

ENVIRONMENT

  AUDIOFILE
	  to get the default host and server to	use.
