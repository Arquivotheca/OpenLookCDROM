NAME
  aevents - An AF event	logging	client.

SYNOPSIS
  aevents [-ringcount count] [-noprint]

DESCRIPTION
  aevents logs all audio server	events to stdout.  The following event types
  are logged:  phone ring, phone DTMF, phone loop current, phone hook switch,
  DSP (if supported by the server), and	property change	events.

Options

  -ringcount count

       Causes aevents to exit after count rings	on the telephone.  Will	not
       exit unless count rings are received as a single	call.

  -noprint

       Supresses normal	event printout.

ENVIRONMENT

  AUDIOFILE	 specifies the host and	server to use.

BUGS
  There	is no way to specify any device	other than device 0.

  If you encounter a reproducible bug, please submit a problem report to
  (af-bugs@crl.dec.com).
