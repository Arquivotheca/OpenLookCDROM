xpow()								       xpow()



Name
  xpow - An AF client to monitor incoming power	levels.

SYNOPSIS
  xpow [-d n] [-log]

Description

  xpow displays	the record level on the	specified audio	device using the
  LineChart widget in the athena widget	set.  All the standard X options
  apply.

  If device not	specified, defaults to first non-telephone device (which is
  usually the local speaker device).

  If -log is used, the power values are	printed	on stdout as they are com-
  puted.
