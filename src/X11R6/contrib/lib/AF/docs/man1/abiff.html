abiff()								      abiff()



Name
  abiff	- An AF	biff play client.

SYNOPSIS
  abiff	[aplay_arg_list]

Description

  abiff	monitors the file /usr/spool/mail for new mail and uses	DECtalk
  text-to-speech synthesis and the AF play client to announce the from and
  subject fields.

  DECtalk software and the tts client used by abiff is not included in the AF
  kit.	However, you can purchase a copy of the	DECtalk	libary for OSF/1.

Options
  All arguments	are passed to aplay.
