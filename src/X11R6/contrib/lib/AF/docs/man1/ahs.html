NAME
  ahs -	An AF hookswitch control client.

SYNOPSIS
  ahs [{on, off}]

DESCRIPTION

  ahs allows you to manipulate the hookswitch on the phone device or deter-
  mine the current state of the	hookswitch.

OPTIONS

  -d d

       Specifies which audio device to use as the phone	device.	 If this
       parameter is not	specified, ahs defaults	to device 0.

  [{on,	off}]

       The hookswitch will be placed in	the state specified by this argument.
       If no argument is given,	ahs prints the current state of	the
       hookswitch.

ENVIRONMENT

  AUDIOFILE	 specifies the host and	server to use.
