NAME
  axset	- AF control panel

SYNOPSIS
  axset

DESCRIPTION
  This program is an X based control panel for an AF server.

  This program is implemented with the Tcl/Tk toolkit, which must be
  installed on your system for the program to run.

RESTRICTIONS
  Doesn't display AF events.


ENVIRONMENT

  AUDIOFILE
	  to get the default host and server to	use.
