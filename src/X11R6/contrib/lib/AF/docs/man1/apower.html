NAME
  apower - Measure signal power	of stdin

SYNOPSIS
  apower [-bs blocksize] [-e encoding] [-ccitt]

DESCRIPTION

  apower measures the power of a signal	on stdin, printing the power level on
  stdout every blocksize.  arecord(1) |	apower is a useful command pipeline.

OPTIONS

  [-bs]

       Specifies the blocksize.	 Defaults to 8000 (one second).

  [-ccitt]

       Calculates the signal power relative to the official CCITT digital
       milliwatt.  Otherwise calculates	the power relative to a	sine wave
       3.16 dB below digital clipping level.

  [-e]

       Specifies the encoding of the data. Valid arguments given by giving a
       random value to -e.

ENVIRONMENT

  none
