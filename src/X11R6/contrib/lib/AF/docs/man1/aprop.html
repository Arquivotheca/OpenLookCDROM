NAME
  aprop	- property displayer for AF

SYNOPSIS
  aprop	[-d device] [-s	device]	[-remove property-name]	[-notype] [-spy]

DESCRIPTION
  The aprop utility is for displaying properties in an AF server.

OPTIONS

  -d device
	  Specifies which audio	device to show properties from.	 If device is
	  not specified, aprop defaults	to device 0.

  -s device
	  Specifies which AF server to use.

  -remove property-name
	  Specifies the	name of	a property to remove.

  -spy	  Examine window properties forever, looking for property change
	  events.

  -notype Specifies that the type of each property should not be displayed.

ENVIRONMENT

  AUDIOFILE	 specifies the host and	server to use.

BUGS
  No attempt is	made to	decode property	values:	 everything is assumed to be
  a zero terminated string.

  If you encounter a reproducible bug, please submit a problem report to
  (af-bugs@crl.dec.com).
