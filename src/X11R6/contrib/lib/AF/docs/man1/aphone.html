NAME
  aphone - An AF phone dialer client.

SYNOPSIS
  aphone [number]

DESCRIPTION

  aphone allows	you to dial the	phone.

Options

  -d d

       Specifies which audio device to use as the phone	device.	 If this
       parameter is not	specified, aphone defaults to device 0.

  -af server

       Specifies which audio server to use.  If	this parameter is not speci-
       fied, aphone defaults to	the value of the AUDIOFILE variable..

  [number]

       Dials the number	specified. Valid characters in the number are 0
       through 9, *, #,	and A through D.  In addition, the comma character
       (,) introduces a	500 millisecond	delay.

ENVIRONMENT

  AUDIOFILE	 specifies the host and	server to use.
