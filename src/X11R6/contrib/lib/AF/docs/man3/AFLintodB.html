NAME
  AFLintodB - decibel to linear

SYNTAX
  #include <AF/AFUtils.h>

  extern double	AFLintodB(double lin);

DESCRIPTION
  AFLintodB is a subroutine for	converting a voltage ratio into	a power	ratio
  expressed in decibels.  The formula is 20.0 *	log10(lin);

ARGUMENTS

  lin  is a voltage ratio.
