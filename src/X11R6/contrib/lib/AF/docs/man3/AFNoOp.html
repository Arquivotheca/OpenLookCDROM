NAME
  AFNoOp - do nothing on an audio connection

SYNTAX
  #include <AF/AFlib.h>

  void AFNoOp(AFAudioConn *aud)

ARGUMENTS

  aud	    Specifies the connection to	the audio server.

DESCRIPTION
  The AFNoOp function does just	that.
