NAME
  AFdBtoLin - convert decibels to linear

SYNTAX
  #include <AF/AFUtils.h>

  extern double	AFdBtoLin(double dB);

DESCRIPTION
  AFdBtoLin is a subroutine for	converting a power ratio in decibels into the
  corresponding	voltage	ratio.	The formula is 10.0 ** (dB/20.0);

ARGUMENTS

  dB   is a power ratio	expressed in decibels.
