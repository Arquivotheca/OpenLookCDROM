AFree(3)							     AFree(3)



NAME
  AFree	- free client data

SYNTAX
  AFree(data)
       void *data;

ARGUMENTS

  data	    Specifies the data that is to be freed.

DESCRIPTION
  The function is a general-purpose AFlib routine that frees the specified
  data.	 You must use it to free any objects that were allocated by AFlib,
  unless an alternate function is explicitly specified for the object.
