NAME
  AFAudioConnName - get	connection name

SYNTAX
  #include <AF/AFlib.h>

  char *AFAudioConnName	(char *conn)

ARGUMENTS

  conn	    Connection name.

DESCRIPTION
  The AFAudioConnName function returns the name	of the connection that
  AFOpenAudioConn would	use.
