Papers

We have written a fairly lengthy technical report on the design principles behind AF, the protocol, client library, server, and performance. A shorter, but less detailed introduction to AudioFile (AF2R2) can be found in our USENIX paper presented at the Summer 93 USENIX. The accompanying PostScript slides are here.

AF and Related Kits

A kit is also available which contains the source code for the AF system including client library, servers for several systems, and device drivers.

crl.dec.com:/pub/DEC/AF
The main distribution site for the latest versions of AF and related documentation. The following are the kit names for AF3.

  • AF3R1.tar.Z
    Released version of AF3R1
  • AF3R1-patches.tar.Z
    Collected set of known patchs to AF3R1.
  • AF2R2-other.tar
    Example sounds bites.
  • ???
    Watch this space for names of Alpha OSF and Ultrix/RISC prebuilt binary setld kits.
Also, we recommend that you retrieve the Tcl/Tk kit as many of our contributed clients depend on this package.
ftp.cs.berkeley.edu:/ucb/tcl
The main distribution site for the latest versions of Tcl, Tk, and related documentation.
We generally ship bug fixes and improvements to the kit as patches which can be applied using Larry Wall's patch utility.
gatekeeper.dec.com:/pub/GNU
One of the distribution sites for the latest version of patch. gcc can also be found here.
